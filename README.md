## Chelsea Fan Blog(Keep the Blue Flag Flying High)
### Client Side
<ul>
    <li>React</li>
    <li>React Hook</li>
    <li>React-Router</li>
</ul>

### Server Side
<ul>
    <li>Node.js</li>
    <li>Express</li>
    <li>MongDB</li>
</ul>

### Feature
<p>첼시 관련 기본 정보 제공</p>
<p>Google map 이용 첼시 스탬포드브릿지 구장 위치 표시</p>
<p>첼시팀 구성원 표시(스쿼드,코치)</p>
<p>첼시 시즌 성적 및 기타 기록 제공 (puppeteer,cheerio 데이터 크롤링 후 DB에 저장)</p>
<p>Chelsea 트윗터 SNS (작업 예정) </p>
